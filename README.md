# AtriLabs-RestauranteX-Homepage

Frontend of a restaurant homepage using Atri framework

## Deployed app


## How to use

### Step 1:

```shell
git clone https://github.com/AmarDiyab/AtriLabs-RestauranteX-Homepage.git

cd AtriLabs-RestauranteX-Homepage

pipenv install atri
```

### Step 2:

```shell
pipenv shell

atri start
```
